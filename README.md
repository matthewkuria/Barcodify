# Barcodify

Modern QR Code & Barcode Generator built with Flask, HTML, CSS and JavaScript.

Create beautiful QR codes and barcodes with custom colors, print layouts, PDF export, PNG download, and batch ZIP generation.

---

## Features

- QR Code Generator
- Barcode Generator
- Modern Glassmorphism UI
- Live Preview
- Dark Mode
- Custom Colors
- PDF Export
- PNG Download
- Print Layout Engine
- Hotel / Restaurant QR Menu Printing
- Batch QR ZIP Export
- Responsive Dashboard
- Drag & Drop Logo Upload
- Flask Backend
- PythonAnywhere Ready
